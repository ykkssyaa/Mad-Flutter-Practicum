# Лабораторная работа 1 — Верстка виджетов

## Как запустить

```powershell
flutter pub get
flutter run
```

## Как проверить

```powershell
flutter test
```

## Что здесь смотреть

- `lib/main.dart` — точка входа приложения
- `lib/app/currency_list/` — экран списка валют
- `test/widget_test.dart` — smoke-тест для проверки запуска UI
