# Proposal: Реализация всех лабораторных работ

Коротко: цель — превратить каждую из шести лабораторных работ в полностью работоспособный Flutter-проект, соответствующий требованиям, описанным в markdown-файлах каждой лабораторной работы.

## Capabilities

- `lab-1-widget-layout` — реализовать ЛР #1 по `lab_1/"Lab 1. Widget layout.md"`
- `lab-2-state-management` — реализовать ЛР #2 по `lab_2/"Lab 2. State management.md"`
- `lab-3-navigation` — реализовать ЛР #3 по `lab_3/"Lab 3. Navigation.md"`
- `lab-4-network` — реализовать ЛР #4 по `lab_4/"Lab 4. Network.md"`
- `lab-5-local-data` — реализовать ЛР #5 по `lab_5/"Lab 5. Local data.md"`
- `lab-6-localization` — реализовать ЛР #6 по `lab_6/"Lab 6. Localization.md"`

Почему это нужно:
- Студент получит шесть завершённых проектов, готовых к запуску и проверке.
- Улучшит качество репозитория: рабочие `main.dart`, корректный `pubspec.yaml`, тесты, CI, и инструкции по запуску.
- Упростит демонстрацию результатов и автоматическое тестирование заданий.

Область работ (ссылки на требования):
- ЛР #1: `lab_1/"Lab 1. Widget layout.md"`
- ЛР #2: `lab_2/"Lab 2. State management.md"`
- ЛР #3: `lab_3/"Lab 3. Navigation.md"`
- ЛР #4: `lab_4/"Lab 4. Network.md"`
- ЛР #5: `lab_5/"Lab 5. Local data.md"`
- ЛР #6: `lab_6/"Lab 6. Localization.md"`

Критерии готовности изменений:
- Для каждой лабораторной работы:
  - Рабочий `lib/main.dart`, который можно запустить (эмулятор/девайс/web).
  - Обновлённый `pubspec.yaml` (зависимости, assets, l10n для ЛР6).
  - Набор интеграционных/виджет-тестов (минимум smoke-тест для старта приложения).
  - Инструкция в `README.md` (как запустить и где смотреть результаты).
- Общие улучшения: статический анализ (`flutter analyze`), CI-шаблон (опционально GitHub Actions), инициализированные локализации для ЛР6.

Ограничения и допущения:
- Подзадачи будут ссылаться на существующие markdown-описания лабораторных работ; детализированную пере-переписку требований не делаем.
- Я не буду менять учебные тексты `Lab *.md`, только исходники проектов и конфигурации для запуска.

Результат:
- Папка `openspec/changes/implement-all-labs/` с артефактами `proposal.md`, `design.md`, `tasks.md`.
