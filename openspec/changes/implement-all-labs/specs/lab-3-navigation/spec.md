## ADDED Requirements

### Requirement: Navigation lab is implemented
The application SHALL implement the navigation scenarios described in `lab_3/"Lab 3. Navigation.md"` as a runnable Flutter app.

#### Scenario: Route navigation works
- **WHEN** the user opens a destination screen from the initial screen
- **THEN** the app navigates to the expected route

### Requirement: Navigation lab is testable
The application SHALL include automated tests that verify at least one navigation path.

#### Scenario: Navigation test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 3 navigation test succeeds
