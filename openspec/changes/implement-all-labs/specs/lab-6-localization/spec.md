## ADDED Requirements

### Requirement: Localization lab is implemented
The application SHALL implement the localization scenarios described in `lab_6/"Lab 6. Localization.md"` as a runnable Flutter app.

#### Scenario: Language changes are reflected
- **WHEN** the app locale changes
- **THEN** the visible strings update to the selected language

### Requirement: Localization lab is testable
The application SHALL include automated tests that verify at least one localized screen state.

#### Scenario: Localization test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 6 localization tests succeed
