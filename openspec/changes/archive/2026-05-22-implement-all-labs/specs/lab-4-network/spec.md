## ADDED Requirements

### Requirement: Network lab is implemented
The application SHALL implement the network scenarios described in `lab_4/"Lab 4. Network.md"` as a runnable Flutter app.

#### Scenario: Remote data is loaded
- **WHEN** the app requests the lab 4 data source
- **THEN** the returned network data is shown in the UI

### Requirement: Network lab is testable
The application SHALL include automated tests that verify the main network-data flow.

#### Scenario: Network test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 4 network-related tests succeed
