## ADDED Requirements

### Requirement: State management lab is implemented
The application SHALL implement the state management scenarios described in `lab_2/"Lab 2. State management.md"` as a runnable Flutter app.

#### Scenario: State changes are reflected in UI
- **WHEN** the user interacts with the lab 2 screen
- **THEN** the visible UI updates according to the current state

### Requirement: State management lab is testable
The application SHALL include automated tests that verify the main state transitions.

#### Scenario: State transition test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 2 state-related tests succeed
