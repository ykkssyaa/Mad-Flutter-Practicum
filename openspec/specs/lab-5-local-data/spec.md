## Requirements
### Requirement: Local data lab is implemented
The application SHALL implement the local storage scenarios described in `lab_5/"Lab 5. Local data.md"` as a runnable Flutter app.

#### Scenario: Data persists locally
- **WHEN** the user saves data in the lab 5 workflow
- **THEN** the data remains available after the app is restarted

### Requirement: Local data lab is testable
The application SHALL include automated tests that verify the local data flow.

#### Scenario: Local storage test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 5 local-data tests succeed
