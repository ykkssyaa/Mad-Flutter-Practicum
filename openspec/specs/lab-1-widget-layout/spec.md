## Requirements
### Requirement: Widget layout lab is implemented
The application SHALL implement the widget layout scenarios described in `lab_1/"Lab 1. Widget layout.md"` as a runnable Flutter app.

#### Scenario: App starts successfully
- **WHEN** the user runs the lab 1 application
- **THEN** the main screen is displayed without runtime errors
- **AND** the layout content matches the lab 1 requirements

### Requirement: Widget layout lab is testable
The application SHALL include at least one automated smoke test for the lab 1 UI.

#### Scenario: Smoke test passes
- **WHEN** the automated test suite is executed
- **THEN** the lab 1 widget test succeeds
